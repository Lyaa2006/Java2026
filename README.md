# 代码/作业互助批改平台（Web 版）

## 运行环境

- Node.js 18+
- 现代浏览器（Chrome、Edge 等）

## 运行说明

在项目根目录执行：

```powershell
cd d:\JavaScript\期末大作业
npx --yes http-server . -p 5173 -c-1
```

浏览器打开：

- http://127.0.0.1:5173/


## 功能一览

- 用户登录与注册，区分学生和教师角色
- 教师创建班级、审批学生加班、发布作业
- 学生加入班级，查看作业并提交作业
- 单次提交支持上传多个文件，并可删除已上传文件
- 教师按提交查看文件，支持下载原始文件、上传批改文件、填写总评
- 文本类文件支持在线批改与在线订正
